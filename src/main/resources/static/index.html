  • [POST] /api/message
    • 쪽지 작성
    • 자기자신에겐 쪽지 못보내게 해야함
    • 탈퇴한 사용자에겐 쪽지 보내지 못함
    • 쪽지보내기 성공 시 새로고침
    • 댓글에서 첫 쪽지 보낼 시 → group: 0, id: comment_id로 생각하여 처리
    • 이미 주고받은 대화에서 쪽지 보낼 시 → group: 1, id: message_id로 생각하여 처리
    • 글에서 첫 쪽지 보냂 시 → group:2, id: board_id로 생각하여 처리
    • 세 경우 모두 comment나 board나 message로 부터 상대방의 user id 를 추출함

    Request

    NameTypecontent
    idintrequest session
    idintrequest body, user_id값은 노출하지 않음 id를 보내서 백에서 user_id값 처리
    contentstringrequest body, 보내는 내용
    groupint0: comment_id / 1: message_id / 2: board_id

    Response

    SUCCESS(200)

    {
	sucess: true,
	data:
			{
				message_id: 1,
			}
}

    FAIL(400)

    // 본인에게 쪽지 보낸 경우
{
	  sucess:false,
	  data: {},
	  message: "자기 자신에게 쪽지를 보낼 수 없습니다"
}

// 탈퇴한 사용자에게 쪽지 보낸 경우
{
	  sucess:false,
	  data: {},
	  message: "탈퇴한 사용자에게 쪽지를 보낼 수 없습니다"
}
